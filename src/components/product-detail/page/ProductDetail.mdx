## Modules break down

Given an specifc address (e.g. _100 madison ave, apt 1707, New York, NY_) and a determined upc (e.g. _00619947000020_) the [product method](https://docs.liquidcommerce.co/references/methods/) fetches from the back-end both the products and retailers availables for thar locality and specific product.

```tsx
const { products, retailers } = await client.product({
  upcs: ['00619947000020'],
  loc: {
    address: {
      one: '100 madison ave',
      two: 'apt 1707',
      city: 'New York',
      state: 'NY',
      zip: '10016',
    },
  },
});
```

Thus, the [response](https://docs.liquidcommerce.co/references/methods/product#response-object) that we obtain (_prouducts_ and _retailers_) feeds the modules:
